# LIFX Console

A [Pry](https://github.com/pry/pry)-enabled REPL to interact with [LIFX](http://lifx.co) devices.

## Features

* Pry-enabled!
* Helpers to help you visually identify lights.

## Installation

$ gem install lifx-console

## Usage

$ lifx-console
